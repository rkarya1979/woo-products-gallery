Woocommerce Products Gallery

== Description ==

Woocommerce Products Gallery is the perfect solution for any WordPress website which needs more flexible options to display woocommerce products in gallery style. This plugin only works with woocommerce.


= Website =
http://www.indianbusybees.com/


== Installation ==

1. Upload 'woo-gallery' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the new menu item "Woo Gallery" and create your first gallery image and assign woocommerce products to this gallery image.
4. Use [wpg_gallery] shortcode to display gallery on any page.
